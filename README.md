GoFileComm — это консольный файловый менеджер, использующий минимум ресурсов процессора.
Из-за того, что код написан в одной основной функции, процессор не тратит ресурсы на вызовы сторонних подпрограмм. Также внедрена цветовая индикация ошибок, чтобы было сразу понятно, что пошло не так.

Функционал:
1. Удаление файлов.
2. Прочтение файлов (для быстрой проверки содержимого).
3. Создание файлов, а также автоматическое добавление расширения .txt, если не указано другое.
4. Создание папок.
5. Просмотр содержимого директорий.
6. Выполнение файлов через системный интерпретатор.

Разделение создания файлов и папок обусловлено удобством: пользователю не нужно дописывать расширения или специальные метки вручную.
